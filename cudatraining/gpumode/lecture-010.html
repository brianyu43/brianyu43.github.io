03 · TERMS용어와 맥락

Glossary3 entries
CUDA — NVIDIA's GPU programming platform and kernel programming model.
kernel — A GPU program launched over many threads/blocks to process data in parallel.
fusion — Combining operations into fewer kernels to reduce memory traffic and launch overhead.
“용어를 외우는 게 아니라, 어떤 병목을 가리키는 이름인지 붙잡는다.”study rule

For CUDA kernels, this lecture is relevant because it touches implementation details such as launch structure, memory movement, profiling, or low-level GPU execution. For Triton kernels, it helps connect Python-level kernel authoring to tiling, program IDs, compiler lowering, and autotuned performance. For ML compilers, it is useful for understanding how higher-level tensor programs become IR, PTX/SASS, scheduled kernels, or fused operators.
